# 3D Maze Runner

[GitHub Repository](https://github.com/Naksh-Rathore/3d-maze-runner)

## Overview
3D Maze Runner is a real-time 3D maze exploration game inspired by classic titles like Doom. It features:

- **Runtime Map Generation**: Procedurally generated mazes for endless replayability.
- **Phong Lighting**: Realistic lighting and shading.
- **Chests & Timed Challenges**: Collect chests within a time limit to score points.
- **Intuitive HUDs**: Clear heads-up displays for player status and progress.

This project is licensed under the **MIT License**. Contributions are welcome and encouraged!

---

## Dependencies
- g++ Compiler
- make utility
- Tested only on **Windows** (using **MinGW**)
- All dependencies must be in your system PATH

---

## Cloning & Building

Clone the repository:

```
git clone https://github.com/Naksh-Rathore/3d-maze-runner.git
cd 3d-maze-runner
```

Build the project using make:

```
make
```

Play!
